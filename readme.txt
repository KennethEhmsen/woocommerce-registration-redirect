After registration, logout the user and redirect to home page in WooCommerce.

== Description ==

WooCommerce Registration Redirect will logout and redirect a user if they enabled account creation on checkout.

== Frequently Asked Questions ==

= What does the plugin do? =

The default in WooCommerce ends up being if create account during checkout is enabled, it will create account for that user and automatically login that user.

== Installation ==

1. Upload the `woocommerce-registration-redirect` directory into `wp-content/plugins/`
2. Activate the plugin through the "Plugins" menu in WordPress
